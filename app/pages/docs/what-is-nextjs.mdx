title: Qu'est-ce que Next.js?
sidebar_label: Qu'est-ce que Next.js?
---

Next.js est intégré dans Blitz et presque toutes les fonctionalités de Next.js sont
supportées. Pour plus d'informations sur les différences, voir ici:
[Pourquoi utiliser Blitz plutôt que Next.js](https://blitzjs.com/docs/why-blitz)

[**Next.js**](https://nextjs.org/) est un Framework React populaire avec beaucoup
 de features intégrées, telles que:

- Pré-rendu, à la fois le rendu de la génération statique \(SSG\) et celui du serveur
  \(SSR\) sont supportés par page
- Découpage automatique du code pour des chargements de page plus rapides
- Routage côté client avec prélecture optimisée
- Environnement de développement qui supporte le remplacement de module à chaud
- Entièrement extensible
